# 나스닥 최종 분류 프롬프트

> 목적: 나스닥 후보 목록을 실제 매수 검토 전 **우선 검토 / 관찰 / 조건 미달 / 제외**로 나눈다.
>
> 핵심 기준: **추세 · 상대강도 · 거래량 · 가격 위치 · 목표가/손절가 · 손익비**

---

## 1. 매수 후보 최종 분류

```
내 나스닥 매수 후보 목록을 오늘 매수 검토용으로 최종 정리해줘.

분류 기준:
1. 추세: 50일선/200일선 위, 정배열 여부
2. QQQ 대비 상대강도
3. 눌림목/돌파/압축 중 어떤 유형인지
4. 거래량 동반 여부
5. 20일선 대비 이격도
6. 목표가와 손절가 기준 손익비
7. 현재 시장 레짐(QQQ 강세 여부)

결과는 네 그룹으로 나눠줘:
1. 우선 검토: 정배열 + 상대강도 강함 + 거래량 동반 + 손익비 2 이상
2. 관찰 필요: 추세는 좋지만 눌림목/거래량 확인 필요
3. 조건 미달: 일부 조건은 좋지만 과열·상대강도·손익비 중 약점 존재
4. 제외: 추세 훼손, 과열, 거래량 급감

각 종목이 왜 그 그룹인지 한 줄로 설명하고, 우선 검토군은 볼 순서대로 정리해줘.
```

## 2. 매수 전 빈칸 확인

```
[종목명]을 매수하기 전에 아직 확인하지 못한 항목을 찾아줘.

확인 항목:
1. 매수 근거 한 문장
2. 추세와 정배열
3. QQQ 대비 상대강도
4. 거래량 동반
5. 목표가
6. 손절가
7. 손익비
8. 시장 레짐
9. 가장 큰 위험 3가지

부족한 항목은 '미확인', '추가 확인 필요', '위험'으로 표시하고,
마지막에 '매수 가능', '관망', '제외'로 결론 내줘.
```

## 3. 3관점 점수화

```
[종목명]을 아래 세 관점에서 각각 0~100점으로 점수화해줘.

1. 추세·차트 강도
2. 실적·성장성
3. 상대강도·거래량

마지막에 종합 매수 매력도를 계산하고,
'매수 가능', '관망', '제외' 중 하나로 결론 내줘.
```

## 4. 목표가·손절가·손익비 최종 확인

```
[종목명]의 현재가 기준 상방 매물대와 하방 지지선을 대조해줘.

계산 항목:
1. 현재가
2. 주요 지지선
3. 손절가
4. 목표가
5. 손실 위험
6. 기대 수익
7. 손익비

손익비가 2 이상이면 '매수 가능',
1.5~2 미만이면 '관망',
1.5 미만이면 '제외'로 결론 내줘.
```

## 5. 나스닥 최종 통합 점검

```
[종목명 또는 후보 목록]을 나스닥 단기 스윙 기준으로 최종 검토해줘.

1. 추세: 50일선/200일선 위, 정배열 여부
2. 상대강도: QQQ 대비 20일 수익률, 하락일 방어력
3. 진입 유형: 눌림목인지, 돌파 전 압축인지, 상투 위험인지
4. 거래량: 20일 평균 대비 증가, 단타성 여부
5. 가격 위치: 20일선 이격, 전고점까지 여유
6. 목표가: 업종 멀티플과 예상 이익 성장 기반
7. 손절가: 20일 ATR과 주요 지지선 이탈 기준
8. 손익비: 2 이상인지
9. 위험 요소 3가지

결과는 '우선 검토', '관찰 필요', '조건 미달', '제외'로 나누고,
종목별 한 줄 결론과 다음 확인 가격대를 제시해줘.
```
