# 나스닥 후보발굴 프롬프트

> 목적: 나스닥 단기 스윙·추세추종 관점에서 새 매수 후보를 찾을 때 사용한다.
>
> 핵심 기준: **QQQ 대비 상대강도 · 20/50/200일선 추세 · 거래량 · 눌림목/돌파 · 손익비**
>
> **자동 스캔:** `python3 screener.py` / `--report` — 아래 §2~§4·§5·§6·§7의 **정량 기준**을 코드로 실행한다.  
> **AI 수동:** §1(섹터 ETF·산업 흐름), 돌파 윗꼬리/종가 위치, 박스권 정의, RR·목표가 최종분류는 `--report` 리포트·`나스닥_최종분류_프롬프트.md`에서 보완.

---

## screener.py 매핑 (프롬프트 ↔ 코드)

| 프롬프트 | screener.py 구현 | 비고 |
|----------|------------------|------|
| §1 주도 섹터 | ❌ 미구현 | ETF·산업 분석은 AI/리서치 수동 |
| §2 눌림목 | ✅ `눌림목` | 50MA↑·정배열·**RS▲**·5MA±2.5%·20MA이격≤8%·RSI 38~65·거래량수축 |
| §3 돌파 | ✅ `돌파` | 52주고점 -6% 이내·RSI 58~82·거래량≥1.1x·200MA↑·**RS▲** |
| §4 에너지응축 | ✅ `에너지응축` | 5-20MA≤5%·20-50MA≤8%·20일레인지≤14%·RSI 43~70·**RS▲** |
| §5 RS | ✅ `rs_ok` | 20일 수익률 > QQQ — **진입 유형 공통 필수** |
| §6 제외 | ✅ 부분 | 50MA↓·시총·부채·펀더(ROE·성장·PEG·PER) — 급등후거래량급감·테마-only는 AI |
| §7 통합 | ✅ + 펀더 | 통과 후 **forward PEG≤2.5** 등 펀더 게이트 추가(프롬프트보다 엄격) |
| 10일선 눌림 | ⚠️ 근사 | 코드는 **5MA±2.5%** + 20MA -4% 허용으로 대체 |
| 손익비·목표가 | ⚠️ 리포트 | `--report` 시 애널 목표·손절·RR 계산 (`nasdaq_buy_report.py`) |

**코드 상수:** `screener.py` 상단 `PB_*`, `BK_*`, `SQ_*`, `ND_*` — 상세는 `기업가치투자기준/스크리너_필터기준_설명.md`

## 1. 주도 섹터 찾기

```
현재 나스닥에서 최근 1개월 동안 자금이 몰리는 섹터 3개를 골라줘.

기준:
1. 섹터 ETF의 상대강도
2. 섹터 지수의 20일선/50일선 배열
3. 대표 대장주와 후발주의 동반 상승 여부
4. 실적 전망 변화
5. AI, 반도체, 전력, 클라우드 등 글로벌 산업 흐름

각 섹터를 '초기 유입', '확산 중', '과열', '관찰 제외'로 나눠줘.
```

## 2. 눌림목 후보 발굴

```
상승 추세에 있는 나스닥 종목 중 10일선 또는 20일선까지 조정받았지만, 거래량이 줄며 지지받는 종목을 찾아줘.

기준:
1. 주가가 20일선 위 또는 근처에서 버티는지
2. 5일선/20일선/50일선 정배열
3. 조정 구간 거래량 감소
4. 52주 고점 대비 하락률
5. QQQ 대비 상대강도 유지
6. 전고점 또는 주요 지지선 방어

결과는 '건강한 눌림목', '관찰 필요', '추세 약화', '매수 보류'로 나눠줘.
```

## 3. 돌파 후보 발굴

```
박스권 상단 또는 52주 신고가를 거래량 급증과 함께 돌파하려는 나스닥 종목을 찾아줘.

기준:
1. 박스권 상단 또는 전고점 가격
2. 현재가와 돌파 가격의 거리
3. 돌파 당일 거래량이 20일 평균 대비 증가했는지
4. 20일선 유지 여부
5. 돌파 후 윗꼬리와 종가 위치
6. 섹터 모멘텀 동반 여부

결과는 '돌파 확인', '돌파 시도', '돌파 실패', '제외'로 나눠줘.
```

## 4. 에너지 응축·삼각수렴

```
현재 주가가 5일선·20일선·50일선 사이에 좁게 갇혀 있거나, 삼각수렴 끝단에 있는 나스닥 종목을 찾아줘.

기준:
1. 최근 20거래일 고점과 저점 폭
2. 5/20/50일선 간격
3. 거래량 축소 후 재확장 조짐
4. 상단 추세선 돌파 가능성
5. 섹터·실적 모멘텀 동반 여부

결과는 '돌파 전 압축', '추가 관찰', '거래대금 부족', '제외'로 나눠줘.
```

## 5. QQQ 대비 상대강도 후보

```
QQQ보다 강한 나스닥 종목을 찾아줘.

기준:
1. 최근 20거래일 수익률이 QQQ보다 높은지
2. 시장 하락일에 낙폭이 더 작았는지
3. 20일선/50일선 위에서 유지되는지
4. 신고가 근접 또는 신고가 경신 여부
5. 거래량 동반 여부

결과는 '주도주', '준주도주', '평범', '약세'로 나눠줘.
```

## 6. 나스닥 후보 제외 필터

```
내 나스닥 후보에서 처음부터 제외할 종목을 골라줘.

제외 기준:
1. 주가가 50일선 또는 200일선 아래에 있어 추세가 훼손됨
2. 단기 급등 후 거래량 급감
3. 20일선 대비 이격 과대
4. 실적 근거 없이 테마로만 급등
5. 변동성이 시장 평균보다 과도하게 큼
6. QQQ 대비 상대강도 약세

결과는 '즉시 제외', '주의 필요', '검토 가능'으로 나눠줘.
```

## 7. 나스닥 후보 통합 발굴

```
나스닥 단기 스윙 후보를 찾아줘.

우선순위:
1. QQQ보다 강한 종목
2. 20일선/50일선 위에서 추세를 유지하는 종목
3. 눌림목 또는 돌파 전 압축 구간에 있는 종목
4. 거래량이 죽지 않고 유지되는 종목
5. 목표가까지 상승 여력이 있고 손절가가 명확한 종목

결과는 '우선 후보', '관찰 후보', '조건 미달', '제외'로 분류하고,
각 종목의 진입 유형과 확인해야 할 가격대를 함께 정리해줘.
```
