# 나스닥 보유·매도 판단 프롬프트

> 목적: 나스닥 보유 종목을 계속 들고 갈지, 비중을 줄일지, 매도할지 판단한다.
>
> 핵심 기준: **손절가 · 목표가 · 추세 이탈 · QQQ 대비 상대강도 · 거래량 · 시장 레짐**

---

## 1. 손절가 도달·추세 이탈 점검

```
내 나스닥 보유 종목 중 추세가 꺾였거나 손절가에 다가간 종목을 찾아줘.

기준:
1. 20일선/50일선 이탈 여부
2. 직전 스윙 저점 이탈 여부
3. 미리 정한 손절가 접근 또는 도달 여부
4. QQQ 대비 상대강도 약화
5. 거래량 동반 하락 여부

결과를 '손절 실행', '이탈 임박', '단순 조정', '문제 없음'으로 나눠줘.
```

## 2. 보유 종목 최종 흐름 분류

```
내 나스닥 보유 종목을 '상승 여지 유지', '보유 재검토', '비중 축소', '매도 우선' 네 단계로 나눠줘.

종합 기준:
1. 목표가까지 남은 상승 여력
2. 추세·정배열 유지 여부
3. QQQ 대비 상대강도
4. 거래량 흐름
5. 20일선 유지 여부
6. 섹터 내 역할 변화
7. 기존 매수 논리 유지 여부

각 종목별 대응을 한 줄로 정리해줘.
```

## 3. 목표가 도달·상승 여력 점검

```
내 나스닥 보유 종목이 목표가에 얼마나 가까워졌는지 점검해줘.

확인 기준:
1. 현재가와 목표가의 거리
2. 목표가 산정 근거가 여전히 유효한지
3. 실적 전망 또는 애널리스트 목표가 변화
4. 최근 상승이 거래량을 동반했는지
5. 단기 과열 또는 윗꼬리 발생 여부

결과는 '보유 유지', '목표가 재산정', '비중 축소', '매도 검토'로 나눠줘.
```

## 4. 촉매 미발생 시 교체 기준

```
매수 후 예상했던 상승 촉매가 나오지 않는 나스닥 종목의 교체 여부를 판단해줘.

기준:
1. 보유 기간
2. 상승 촉매 발생 여부
3. 거래량 유지 여부
4. QQQ 대비 상대강도
5. 20일선 유지 여부
6. 더 나은 다른 후보의 매력도

결과는 '보유 유지', '관찰 연장', '비중 축소', '교체 검토'로 나눠줘.
```

## 5. 시장 위험 단계 점검

```
현재 나스닥 시장을 '정상 확장', '참여도 축소', '투기 확산', '위험 관리' 네 단계 중 하나로 분류해줘.

판단 기준:
1. 상승 종목 수
2. 대형주 소수 의존도
3. 52주 신고가 종목 변화
4. VIX 등 변동성 확대
5. 20일선 이탈 종목 수
6. 저품질 종목 급등 여부

'위험 관리' 단계면 신규 진입을 줄이고 손절가 관리를 강화하라고 조언해줘.
```

## 6. 방어력·방어 착시 점검

```
최근 시장 조정 구간에서 내 나스닥 관심·보유 종목의 방어력을 점검해줘.

구분:
1. 진짜 강한 종목: 지수 하락일 낙폭이 작고, 20일선 유지, 거래량 유지
2. 방어 착시: 거래량 급감 또는 시장 관심 감소로 단지 덜 움직이는 종목

결과를 '강함', '관찰', '방어 착시', '제외'로 나눠줘.
```

## 7. 매매 복기

```
오늘의 나스닥 매매를 복기해줘.

포함 항목:
1. 매수/매도 종목과 판단 근거
2. 진입 유형(눌림목/돌파/추격 매수)
3. 목표가와 손절가
4. 손익비
5. 반복되는 실수(추격 매수, 손절 지연, 거래량 부족 종목 진입 등)
6. 다음 점검 날짜

과거 손실 패턴이 반복됐다면 다음 진입 때 띄울 경고 문구도 만들어줘.
```
