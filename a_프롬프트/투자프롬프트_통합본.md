## 0. 내 투자 전략 기준

나는 시장별로 다른 전략을 쓴다.

| 시장 | 전략 | 핵심 기준 |
|------|------|-----------|
| 나스닥 | 단기 스윙·추세추종 | 추세, 상대강도, 거래량, 눌림목/돌파, 목표가·손절가 |
| 코스피 | 가치투자·자산 방어 | 안전마진, 재무 안정성, 배당, 저변동, 기관/외국인 수급 |

따라서 모든 프롬프트는 먼저 **나스닥용인지, 코스피용인지** 구분해서 사용한다.

---

## 1. 전체 사용 흐름

프롬프트는 한 번에 전부 쓰지 않는다. 아래 순서대로 필요한 것만 골라 쓴다.

| 단계 | 목적 | 핵심 질문 |
|------|------|-----------|
| 1단계 | 시장·업종 확인 | 지금 돈이 몰리는 산업인가? |
| 2단계 | 후보 발굴 | 차트·실적·수급 중 무엇이 좋아 보이는가? |
| 3단계 | 제외 필터 | 처음부터 사면 안 되는 종목인가? |
| 4단계 | 재무·실적 점검 | 실적이 실제로 좋아지고 있는가? |
| 5단계 | 수급·거래대금 확인 | 큰 자금이 들어오고 유지되는가? |
| 6단계 | 가격 위치 확인 | 지금 사도 되는 자리인가? |
| 7단계 | 매매 시나리오 | 목표가·손절가·손익비가 맞는가? |
| 8단계 | 최종 분류 | 바로 검토, 관찰, 제외 중 어디인가? |
| 9단계 | 보유 관리 | 계속 들고 갈 근거가 남아 있는가? |

## 2. 시장·주도 산업 찾기

### 주도 산업 분석 프롬프트

```
최근 1개월 동안 시장에서 자금이 몰리는 산업 3개를 골라줘.

기준:
1. 산업별 거래대금 증가율
2. 관련 ETF 순자산 변화
3. 기관/외국인 수급
4. 대표 기업과 후발 종목의 동반 상승 여부
5. 실적 전망 변화
6. 정책 또는 글로벌 산업 흐름

각 산업을 '초기 유입', '확산 중', '과열', '관찰 제외'로 나눠줘.
```

### 업종 확산 수급 프롬프트

```
최근 한 종목에만 수급이 몰린 것이 아니라, 같은 업종 여러 기업으로 기관/외국인 매수가 퍼지는 산업을 찾아줘.

비교 대상:
1. 업종 대표주
2. 후발주
3. 관련 ETF
4. 최근 20거래일 수급
5. 최근 20거래일 거래대금 변화

현재 단계가 '대장주 선행', '후발주 확산', 'ETF 자금 유입', '확산 둔화' 중 어디인지 정리해줘.
```

### 대장주와 후발주 비교 프롬프트

```
섹터 내 대장주는 신고가를 경신했지만, 재무가 우량함에도 아직 덜 오른 후발 종목을 찾아줘.

기준:
1. 대장주 대비 주가 상승률
2. 실적 성장률
3. 밸류에이션 부담
4. 기관/외국인 수급
5. 업종 내 역할
6. 대장주 하락 시 방어력

결과는 '후발 키맞추기 후보', '대장주만 강함', '업종 확산 부족', '제외'로 나눠줘.
```

## 3. 후보 발굴: 차트의 급소와 급등 전 신호

### 눌림목 후보 발굴 프롬프트

```
상승 추세에 있는 종목 중 10일선 또는 20일선까지 조정받았지만, 거래량이 줄며 지지받는 종목을 찾아줘.

기준:
1. 주가가 20일선 위 또는 근처에서 버티는지
2. 5일선/20일선/60일선 배열
3. 조정 구간 거래량 감소
4. 최근 고점 대비 하락률
5. 기관/외국인 수급 유지
6. 전고점 또는 주요 지지선 방어

결과는 '건강한 눌림목', '관찰 필요', '추세 약화', '매수 보류'로 나눠줘.
```

### 돌파 후보 발굴 프롬프트

```
최근 3개월간 특정 가격대에 갇혀 있다가 박스권 상단 또는 주요 매물대를 강하게 돌파하려는 종목을 찾아줘.

기준:
1. 박스권 상단 가격
2. 현재가와 상단 가격의 거리
3. 돌파 당일 또는 최근 거래대금 증가
4. 20일선 유지 여부
5. 돌파 후 윗꼬리와 종가 위치
6. 실적 또는 업종 모멘텀 동반 여부

결과는 '돌파 확인', '돌파 시도', '돌파 실패', '제외'로 나눠줘.
```

### 에너지 응축·삼각수렴 프롬프트

```
현재 주가가 5일선·20일선·60일선 사이에 좁게 갇혀 있거나, 고점은 낮아지고 저점은 높아지는 삼각수렴 끝단에 있는 종목을 찾아줘.

기준:
1. 최근 20거래일 고점과 저점 폭
2. 5/20/60일선 간격
3. 거래대금 유지 여부
4. 상단 추세선 돌파 가능성
5. 수급 또는 실적 모멘텀 동반 여부

결과는 '돌파 전 압축', '추가 관찰', '거래대금 부족', '제외'로 나눠줘.
```

### 거래량 선행 흐름 프롬프트

```
최근 20거래일 동안 거래량 또는 거래대금은 늘었지만 주가 상승폭은 크지 않은 종목을 찾아줘.

조건:
1. 거래량/거래대금이 20거래일 평균보다 증가
2. 주가 상승률은 제한적
3. 기관/외국인 수급 변화가 함께 나타남
4. 개인 매수만 몰린 종목은 따로 표시

각 종목을 '의미 있는 자금 유입', '개인 매수 쏠림', '돌파 전 대기', '관찰 제외'로 나눠줘.
이후 확인해야 할 가격 구간도 정리해줘.
```

## 4. 처음부터 제외할 종목 필터

### 제외 조건 정리 프롬프트

```
내 매수 후보에서 처음부터 제외해야 할 종목을 골라줘.

제외 기준:
1. 지속 적자 또는 실적 개선 근거 부족
2. 과도한 부채 또는 재무 부담
3. 영업현금흐름 악화
4. 평균 거래대금 부족
5. 단기 급등 후 거래량 감소
6. 전환사채, 유상증자, 오버행 가능성
7. 뉴스 기대감은 크지만 실적이 따라오지 않는 종목
8. 개인 매수 쏠림이 강하고 기관/외국인 참여가 약한 종목

결과는 '즉시 제외', '주의 필요', '검토 가능'으로 나눠줘.
각 종목이 왜 해당 그룹에 들어갔는지 한 줄로 설명해줘.
```

### 가격 기대감 과반영 필터

```
현재 주가에 시장 기대감이 과도하게 반영된 종목을 걸러줘.

판단 기준:
1. 최근 1개월/3개월 급등률
2. 거래대금 급증 후 감소 여부
3. 목표주가 상향 후 주가가 이미 크게 반응했는지
4. 실적 대비 PER/PBR/PSR 부담
5. 뉴스 노출 증가와 개인 매수 쏠림

이미 가격에 반영된 종목은 '가격 부담'으로 표시하고 제외 이유를 설명해줘.
```

### 고변동·고위험 차단 프롬프트

```
나는 원금 회복보다 자산 방어를 우선한다.
변동성이 시장 평균보다 1.5배 높거나, 손절 기준을 쉽게 흔드는 고위험 종목은 제외해줘.

기준:
1. 최근 20일 평균 변동폭
2. 지수 대비 변동성
3. 급등락 반복 여부
4. 거래대금 감소와 변동성 확대 동시 발생 여부
5. 실적보다 테마로 움직이는지

결과는 '투자 가능', '변동성 주의', '고위험 제외'로 나눠줘.
```

## 5. 재무·실적·밸류에이션 점검

### 재무 안정성 점검 프롬프트

```
내 매수 후보 목록의 재무 상태를 점검해줘.

기준:
1. 최근 매출 성장률
2. 영업이익 흑자 여부
3. 영업이익률 변화
4. 부채비율
5. 영업현금흐름
6. 최근 4개 분기 실적 개선 여부
7. 일회성 이익 포함 여부

결과는 '통과', '추가 확인', '재무 부담', '탈락'으로 나눠줘.
```

### 실적 개선 중심 종목 선별

```
단기 테마주보다 실적 성장 가능성이 높은 기업 중심으로 정리해줘.

우선 조건:
1. 최근 매출 증가
2. 영업이익 증가
3. 영업이익률 개선
4. 기관/외국인 수급 지속성
5. 업종 성장 흐름이 숫자로 확인됨

실적이 아닌 테마와 기대감만으로 오른 종목은 제외해줘.
```

### 이익 전망 상향 프롬프트

```
최근 3개월 동안 영업이익 전망치가 꾸준히 상향된 기업을 추려줘.

비교 항목:
1. 추정치 상향 횟수
2. 상향 폭
3. 매출 전망 변화
4. 영업이익률 개선 여부
5. 목표주가 변화

전망 상향이 일회성인지, 추세적인 변화인지 구분해줘.
```

### 저평가·안전마진 프롬프트

```
현재 커뮤니티나 시장에서 공포가 큰 섹터를 찾고, 해당 섹터 우량주 중 역사적 저평가 구간에 진입한 종목을 골라줘.

기준:
1. PER/PBR/PSR이 과거 평균보다 낮은지
2. 업종 평균 대비 저평가인지
3. 매출과 영업이익이 훼손되지 않았는지
4. 영업현금흐름이 순이익보다 안정적인지
5. 부채 구조와 배당 지속성이 괜찮은지

결과는 '안전마진 있음', '저평가처럼 보이나 위험', '추가 확인', '제외'로 나눠줘.
```

## 6. 수급·거래대금·큰손 확인

### 거래대금 증가 확인 프롬프트

```
내 매수 후보 중 최근 거래대금이 충분하고 실제로 증가하는 종목만 골라줘.

확인 기준:
1. 최근 5거래일 평균 거래대금
2. 최근 20거래일 평균 거래대금
3. 최근 3개월 평균 대비 거래대금 증가율
4. 거래대금 증가가 주가 상승과 함께 나타났는지
5. 하루짜리 단타성 거래인지, 여러 날 이어지는 관심인지
6. 급등 후 거래대금이 줄어드는 위험 구간인지

결과는 '의미 있는 관심 증가', '단기 급등 후 약화', '거래대금 부족', '제외'로 나눠줘.
```

### 기관·외국인 수급 지속성 프롬프트

```
내 매수 후보 중 기관과 외국인 수급이 여러 날 이어지는 종목을 찾아줘.

하루 순매수보다 아래 흐름을 더 중요하게 봐줘:
1. 최근 5거래일 누적 순매수
2. 최근 10거래일 누적 순매수
3. 최근 20거래일 누적 순매수
4. 중간에 강한 매도 전환이 있었는지
5. 수급이 주가 상승과 함께 이어지는지
6. 단기 급등 후 뒤늦게 들어온 수급인지

결과는 '수급 지속', '수급 초기', '수급 약화', '주의'로 나눠줘.
```

### 큰 자금 초기 진입 판별 프롬프트

```
[종목명]이 큰 자금의 초기 진입 구간인지, 이미 시장 관심이 과도하게 붙은 후반 구간인지 판단해줘.

판단 기준:
1. 최근 20거래일 거래대금 변화
2. 저점 상승 여부
3. 고점 압축 구간 여부
4. 장중 하락 후 종가 회복 패턴
5. 지수 하락일 방어력
6. 뉴스 노출 정도
7. 개인 매수 쏠림 여부
8. 기관/외국인 수급 지속성

결과는 '초기 자금 유입', '돌파 전 압축', '후반 관심 과열', '관찰 제외'로 나눠줘.
```

### 수급 채우기·숏스퀴즈 프롬프트

```
업황 회복 조짐은 뚜렷하지만 기관·외국인 보유 비중이 낮아 수급 채우기가 예상되는 종목을 찾아줘.

추가로 주가는 반등하는데 공매도 잔고가 빠르게 줄어드는 종목이 있다면 숏스퀴즈 가능성을 따로 표시해줘.

기준:
1. 기관·외국인 보유 비중의 과거 대비 위치
2. 최근 순매수 전환 여부
3. 공매도 잔고 변화
4. 거래대금 증가
5. 실적 전망 유지 여부

결과는 '수급 채우기 후보', '숏스퀴즈 후보', '관찰 필요', '제외'로 나눠줘.
```

## 7. 가격 위치·눌림목·상투 판단
