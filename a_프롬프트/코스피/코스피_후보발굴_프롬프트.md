# 코스피 후보발굴 프롬프트

> 목적: 코스피 가치투자 관점에서 새 매수 후보를 찾을 때 사용한다.
>
> 핵심 기준: **안전마진 · 재무 안정성 · 실적 모멘텀 · 수급 전환 · 목표주가 괴리율**
>
> 방향: 배당 방어주에만 집중하지 않는다. 실적 개선·업황 반등·업종 내 키맞추기 종목도 함께 탐색한다.

---

## 1. 공포 구간 저평가 + 낙폭 과대 섹터 찾기

```
현재 시장에서 공포 또는 급락으로 낙폭이 과도해진 섹터를 찾고,
그 안에서 실적과 재무가 훼손되지 않은 코스피 우량주를 골라줘.

기준:
1. PER/PBR이 과거 평균보다 낮은지 (역사적 저평가 여부)
2. 업종 평균 대비 저평가인지
3. 매출과 영업이익이 훼손되지 않았는지
4. 12개월 선행 EPS 추정치가 상향 또는 유지되는지
5. 20일 이격도가 90% 이하인지 (낙폭 과대 여부)
6. 외국인·기관 순매수가 재개되는 조짐이 있는지

결과는 '안전마진+낙폭 과대', '저평가만', '낙폭 과대만', '제외'로 나눠줘.
```

## 2. 어닝 서프라이즈·실적 모멘텀 종목 탐색

```
최근 또는 다음 실적 발표에서 시장 예상을 크게 웃도는 실적이 기대되는 코스피 종목을 찾아줘.

기준:
1. 최근 분기 영업이익이 컨센서스를 10% 이상 초과했거나 초과가 예상되는지
2. 영업이익 증가율 (YoY)이 급등하는지
3. 증권사 목표주가가 최근 상향 조정됐는지
4. 현재가와 컨센서스 목표주가의 괴리율이 20% 이상인지
5. 실적 개선 원인이 일회성이 아닌 구조적인지

결과는 '강한 모멘텀', '모멘텀 초기', '일회성 개선', '제외'로 나눠줘.
```

## 3. 그룹·업종 내 키맞추기 후보 찾기

```
같은 그룹 또는 업종에서 대장주는 많이 올랐지만, 아직 덜 오른 후발 종목을 찾아줘.

기준:
1. 대장주 대비 주가 상승률 차이 (연초比 또는 최근 3개월)
2. 재무·실적이 대장주와 비슷하게 좋거나 오히려 더 나은지
3. 목표주가 대비 현재가 괴리율이 더 큰지
4. PBR·PER이 대장주보다 낮은지
5. 외국인·기관 수급이 이동하는 조짐이 있는지

결과는 '키맞추기 유력 후보', '수급 이동 초기', '업종 확산 부족', '제외'로 나눠줘.
```

## 4. 재무 안정성 점검

```
내 코스피 후보의 재무 안정성을 점검해줘.

기준:
1. 최근 매출 성장률
2. 영업이익 흑자 지속 및 개선 방향
3. 영업이익률 변화
4. 부채비율 200% 이하 여부
5. 영업현금흐름의 안정성
6. 최근 4개 분기 실적 개선 여부
7. 일회성 이익 포함 여부

결과는 '통과', '추가 확인', '재무 부담', '탈락'으로 나눠줘.
```

## 5. 외국인·기관 수급 전환 탐색

```
최근 외국인 또는 기관이 순매수로 전환하거나 순매수가 이어지는 코스피 종목을 찾아줘.

하루 순매수보다 아래 흐름을 더 중요하게 봐줘:
1. 최근 5거래일 누적 순매수 규모
2. 최근 10거래일·20거래일 누적 순매수
3. 시장 급락 구간에서 외국인·기관이 저가 매수에 나선 종목
4. 수급 전환이 주가 하락 중에 시작됐는지 (진짜 초기 매집인지)
5. 단기 급등 후 뒤늦게 들어온 수급인지

결과는 '수급 초기 진입', '수급 지속', '수급 약화', '주의'로 나눠줘.
```

## 6. 목표주가 괴리율 큰 종목 탐색

```
현재 주가와 증권사 컨센서스 목표주가의 괴리율이 큰 코스피 종목을 찾아줘.

기준:
1. 컨센서스 목표주가 대비 현재가 괴리율 20% 이상
2. 최근 목표주가가 상향 조정됐는지 (상향 조정 = 추가 상승 여력 의미)
3. 목표주가 상향 이유가 실적 개선인지, 단순 센티멘트인지 구분
4. 증권사 투자의견 매수(BUY) 비율이 70% 이상인지
5. 목표주가 도달 시나리오가 현실적인지

결과는 '강한 상승여력', '보통', '목표가 괴리 과소', '제외'로 나눠줘.
```

## 7. 처음부터 제외할 종목 필터

```
내 코스피 후보에서 처음부터 제외할 종목을 골라줘.

제외 기준:
1. 지속 적자 또는 실적 개선 근거 부족
2. 과도한 부채 또는 재무 부담
3. 영업현금흐름 악화
4. 평균 거래대금 부족
5. 전환사채, 유상증자, 오버행 가능성
6. 뉴스 기대감만 크고 실적이 따라오지 않는 종목
7. 개인 매수 쏠림, 기관/외국인 참여 약함

결과는 '즉시 제외', '주의 필요', '검토 가능'으로 나누고 이유를 한 줄로 설명해줘.
```

## 8. 배당·안전마진 후보 (선택 적용)

```
배당과 안전마진을 동시에 갖춘 코스피 종목을 골라줘.
단, 배당이 없더라도 실적 모멘텀과 수급이 좋다면 후보에서 제외하지 않는다.

기준:
1. 배당수익률과 배당 지속성 (필수가 아닌 가산점)
2. 배당성향이 과도하지 않은지
3. PBR·PER이 과거 평균과 업종 평균 대비 낮은지
4. 순현금 또는 낮은 순부채
5. 하락장에서 방어력

결과는 '배당+안전마진', '안전마진만', '모멘텀 중심', '해당 없음'으로 나눠줘.
```

## 9. 코스피 후보 통합 발굴

```
코스피 매수 후보를 찾아줘.

우선순위:
1. 실적 모멘텀이 좋고, 컨센서스 목표주가 대비 괴리율이 20% 이상인 종목
2. 낙폭이 과대하지만 12개월 선행 EPS가 상향되고 외국인·기관 수급이 들어오는 종목
3. 같은 그룹·업종 내 대장주 대비 덜 오른 키맞추기 후보
4. 재무가 안정적이고 안전마진이 있는 종목
5. 배당은 가산점. 없어도 위 조건을 충족하면 후보에 포함한다.

절대 제외:
- 지속 적자, 오버행, 거래대금 부족, 기관/외국인 전혀 없는 종목

결과는 '우선 후보', '관찰 후보', '조건 미달', '제외'로 분류하고,
각 종목의 핵심 투자 근거 한 줄 + 가장 큰 리스크 한 줄로 정리해줘.
```
