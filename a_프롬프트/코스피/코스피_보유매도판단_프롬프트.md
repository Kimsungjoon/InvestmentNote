# 코스피 보유·매도 판단 프롬프트

> 목적: 코스피 보유 종목을 계속 들고 갈지, 비중을 줄일지, 매도할지 판단한다.
>
> 핵심 기준: **안전마진 유지 · 재무 훼손 · 배당 지속성 · 기관/외국인 수급 · 자산 방어**

---

## 1. 수급 이탈 경고

```
내 코스피 보유 종목 중 기관과 외국인의 매수 강도가 약해진 종목을 찾아줘.

경고 기준:
1. 20거래일 누적 순매수에서 5거래일 순매도로 전환
2. 거래대금 감소
3. 20일선 이탈
4. 주가가 수급보다 먼저 약해짐
5. 업종 내 상대 순위 하락

결과를 '수급 이탈 초기', '매도세 본격화', '단순 조정', '문제 없음'으로 나눠줘.
```

## 2. 보유 종목 최종 흐름 분류

```
내 코스피 보유 종목을 '상승 여지 유지', '보유 재검토', '비중 축소', '매도 우선' 네 단계로 나눠줘.

종합 기준:
1. 안전마진이 아직 남았는지
2. 재무·실적 훼손 여부
3. 배당 지속성
4. 기관/외국인 수급 유지
5. 변동성 확대 여부
6. 최초 매수 근거 유지 여부
7. 자산 방어 원칙에 맞는지

각 종목별 대응을 한 줄로 정리해줘.
```

## 3. 손절·비중 축소 우선순위

```
내 코스피 보유 종목 중 손절 또는 비중 축소를 먼저 검토할 순서를 정해줘.

기준:
1. 안전마진 소멸
2. 실적 또는 현금흐름 훼손
3. 회사의 실적 설명 변화
4. 회복 시점 지연
5. 수급 이탈
6. 거래대금 감소
7. 본전 집착 위험

결과는 '우선 정리 후보', '관찰 후 축소 후보', '보유 가능 후보'로 나눠줘.
```

## 4. 목표가·가치 회복 점검

```
내 코스피 보유 종목이 목표가에 얼마나 가까워졌는지, 가치 회복 논리가 아직 유효한지 점검해줘.

확인 기준:
1. 현재가와 목표가의 거리
2. 목표가 산정 근거가 여전히 유효한지
3. 실적 전망 변화
4. 배당 정책 변화
5. 기관/외국인 수급 유지 여부
6. 저평가 해소 여부

결과는 '보유 유지', '목표가 재산정', '비중 축소', '매도 검토'로 나눠줘.
```

## 5. 포트폴리오 집중도·분산 점검

```
내 코스피 포트폴리오가 특정 섹터에 30% 이상 쏠려 있는지 확인해줘.

쏠림이 있다면:
1. 해당 섹터 비중
2. 같은 방향으로 움직일 위험
3. 상관관계가 낮은 대체 섹터
4. 방어형 우량주 또는 배당 ETF 후보

결과는 '균형 양호', '쏠림 주의', '분산 필요'로 나눠줘.
```

## 6. 자산 방어 관점 위험 점검

```
내 코스피 보유 종목 중 원금 회복 기대보다 자산 방어 원칙을 우선해야 하는 종목을 찾아줘.

기준:
1. 손실 중이지만 회복 근거가 약한지
2. 실적 회복 시점이 계속 지연되는지
3. 수급이 이탈했는지
4. 배당 또는 재무 안정성이 훼손됐는지
5. 더 안정적인 대체 후보가 있는지

결과는 '보유 가능', '비중 축소', '정리 우선'으로 나눠줘.
```

## 7. 매매 복기

```
오늘의 코스피 매매를 복기해줘.

포함 항목:
1. 매수/매도 종목과 판단 근거
2. 안전마진·재무 근거
3. 목표가와 손절가
4. 자산 방어 원칙을 지켰는지
5. 반복되는 실수(본전 집착, 손절 지연, 테마 의존 등)
6. 다음 점검 날짜

과거 손실 패턴이 반복됐다면 다음 진입 때 띄울 경고 문구도 만들어줘.
```
