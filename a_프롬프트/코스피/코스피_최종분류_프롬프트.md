# 코스피 최종 분류 프롬프트

> 목적: 코스피 후보 목록을 실제 매수 검토 전 **우선 검토 / 관찰 / 조건 미달 / 제외**로 나눈다.
>
> 핵심 기준: **안전마진 · 재무 안정성 · 배당 · 기관/외국인 수급 · 저변동**

---

## 1. 매수 후보 최종 분류

```
내 코스피 매수 후보 목록을 매수 검토용으로 최종 정리해줘.

분류 기준:
1. 제외 조건 여부(적자, 오버행, 고변동)
2. 재무 안정성(부채비율, 영업현금흐름)
3. 안전마진(밸류에이션 저평가)
4. 배당 지속성
5. 기관/외국인 수급 지속성
6. 변동성(저변동 우선)
7. 실적 개선 방향

결과는 네 그룹으로 나눠줘:
1. 우선 검토: 안전마진 + 재무 안정 + 수급 지속 + 저변동
2. 관찰 필요: 밸류는 좋으나 수급 또는 실적 확인 필요
3. 조건 미달: 일부 좋지만 재무/변동성/수급 약점 존재
4. 제외: 재무 부담, 고변동, 오버행, 유동성 부족

각 종목이 왜 그 그룹인지 한 줄로 설명하고, 우선 검토군은 볼 순서대로 정리해줘.
```

## 2. 매수 전 빈칸 확인

```
[종목명]을 매수하기 전에 아직 확인하지 못한 항목을 찾아줘.

확인 항목:
1. 매수 근거 한 문장
2. 안전마진(저평가 근거)
3. 재무 안정성(부채, 현금흐름)
4. 배당
5. 기관/외국인 수급 지속성
6. 변동성 수준
7. 손절 기준(자산 방어선)
8. 업종 흐름
9. 가장 큰 위험 3가지

부족한 항목은 '미확인', '추가 확인 필요', '위험'으로 표시하고,
마지막에 '매수 가능', '관망', '제외'로 결론 내줘.
```

## 3. 3관점 점수화

```
[종목명]을 아래 세 관점에서 각각 0~100점으로 점수화해줘.

1. 재무·안전마진
2. 수급의 연속성
3. 변동성·방어력

마지막에 종합 매수 매력도를 계산하고,
'매수 가능', '관망', '제외' 중 하나로 결론 내줘.
```

## 4. 목표가·손절가 최종 확인

```
[종목명]의 가치투자 관점 목표가와 자산 방어용 손절가를 계산해줘.

목표가 기준:
1. 과거 평균 PER/PBR/PSR
2. 업종 평균 밸류에이션
3. 예상 이익과 배당 지속성
4. 무리한 낙관이 아닌 현실적 회복 가격

손절가 기준:
1. 실적 훼손 여부
2. 주요 지지선 이탈
3. 기관/외국인 수급 이탈
4. 변동성 확대
5. 최초 매수 근거 훼손

출력: 현재가 / 목표가 / 손절가 / 안전마진 / 자산 방어 리스크 / 매수 가능 여부.
```

## 5. 코스피 최종 통합 점검

```
[종목명 또는 후보 목록]을 코스피 가치투자·자산 방어 기준으로 최종 검토해줘.

1. 제외 조건: 적자, 부채 부담, 현금흐름 악화, 유동성 부족, 오버행, 고변동
2. 재무: 최근 4개 분기 매출/영업이익/영업이익률, 부채비율 200% 이하
3. 안전마진: PER/PBR/PSR이 과거·업종 평균 대비 저평가인지
4. 배당: 배당수익률과 지속성
5. 수급: 최근 5/10/20거래일 기관·외국인 누적 순매수와 매도 전환 여부
6. 변동성: 지수 대비 저변동인지
7. 자산 방어선: 흔들림엔 견디되 훼손 시 정리할 손절가

결과는 '우선 검토', '관찰 필요', '조건 미달', '제외'로 나누고,
종목별 한 줄 결론과 다음 확인 가격대를 제시해줘.
```
