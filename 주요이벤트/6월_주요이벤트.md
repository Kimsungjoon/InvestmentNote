# 2026년 6월 주요 경제 이벤트 — 쉬운 설명

> 이 문서는 6월에 주가를 흔들 수 있는 핵심 이벤트(CPI·FOMC 등)를 **투자 초보도 이해할 수 있게** 풀어 쓴 것입니다.
> 마지막에 내 보유 종목(ANET·NBIS·AMD·AMAT)과의 연관성도 정리했습니다.

---

## 0. 먼저, 왜 이런 지표가 주가를 흔드나?

대부분의 흐름은 **"금리"** 한 단어로 연결됩니다.

```
물가가 높다 → 연준이 금리를 높게 유지 → 빌린 돈이 비싸짐
→ 미래 이익으로 평가받는 "성장주"가 불리 → 주가 하락

물가가 안정 → 금리 인하 기대 → 돈이 싸짐
→ 성장주에 유리 → 주가 상승
```

> 우리 보유 종목(AI·반도체)은 전부 **"성장주"** 라서, 금리 방향에 특히 민감합니다.
> 그래서 **물가 지표(CPI)**와 **금리를 정하는 회의(FOMC)**가 6월 최대 변수입니다.

---

## 1. CPI (소비자물가지수) — "물가가 얼마나 올랐나"

### 한 줄 정의
> **CPI = Consumer Price Index = 소비자물가지수.**
> 일반 가정이 사는 물건·서비스(식품·집세·교통·의료 등) 가격이 **1년 전보다 얼마나 올랐는지**를 % 로 나타낸 숫자.

### 쉬운 비유
```
작년에 장바구니 한가득 = 100,000원
올해 똑같은 장바구니   = 103,000원
→ CPI 상승률 = +3%  ("물가가 3% 올랐다")
```

### 왜 주가에 중요한가?
- 연준(미국 중앙은행)은 **물가를 잡는 게 임무**입니다. 목표는 보통 **연 2%**.
- **CPI가 예상보다 높게 나오면** → "금리를 더 오래 높게 둬야겠네" → 성장주 **하락**
- **CPI가 예상보다 낮게 나오면** → "이제 금리 내려도 되겠다" → 성장주 **상승**

> 핵심은 "절대 수치"보다 **시장 예상치(컨센서스) 대비 높냐/낮냐**입니다.
> 예: 예상 2.8% → 실제 3.2%면 "쇼크"로 급락할 수 있음.

### 6월 일정
- **5월 CPI 발표: 6월 10일 (미국 오전 8:30 ET)**

## 2. FOMC — "금리를 정하는 회의"

### 한 줄 정의
> **FOMC = Federal Open Market Committee = 연방공개시장위원회.**
> 미국 중앙은행(연준, Fed)이 **기준금리를 올릴지/내릴지/동결할지 결정하는 회의.**

### 쉬운 비유
```
FOMC = 나라 경제의 "온도 조절기"를 만지는 사람들

경제가 과열(물가↑) → 금리를 올려 식힌다 (성장주에 찬물)
경제가 냉각(침체)   → 금리를 내려 데운다 (성장주에 훈풍)
```

### 무엇을 보나? (3가지)
1. **금리 결정**: 올림/내림/동결 — 가장 직접적
2. **점도표(SEP)**: 위원들이 생각하는 **앞으로의 금리 전망** 점 그래프 → "올해 몇 번 더 내릴 생각인가"를 보여줌
3. **파월 의장 기자회견**: 말 한마디(매파적=긴축/비둘기파=완화)에 따라 시장이 출렁

### 매파 vs 비둘기파 (꼭 알아두기)
```
🦅 매파(Hawkish)  = "물가 잡으려면 금리 높게!" → 긴축 → 주가에 악재
🕊️ 비둘기파(Dovish) = "경제 살리게 금리 낮추자" → 완화 → 주가에 호재
```

### 6월 일정
- **FOMC 회의: 6월 16~17일** (이틀)
- **결과 발표 + 점도표 + 파월 회견: 6월 17일 (미국 오후 2:00 ET)**

## 5. 내 보유 종목과의 연관성

> 보유: **ANET(AI 네트워킹) · NBIS(AI 클라우드) · AMD(AI 반도체) · AMAT(반도체 장비)**
> → 4종목 모두 **AI·반도체 성장주**라 같은 이벤트에 함께 움직입니다.

| 이벤트 | 영향 종목 | 어떻게 |
|--------|----------|--------|
| **CPI(6/10)** | 전 종목 | 물가 높으면 동반 하락, 특히 NBIS·AMD(고PER) |
| **FOMC(6/17)** | 전 종목 | 매파적이면 급락, 비둘기파면 랠리 |
| Computex(6/2~5) | AMD·AMAT | 신제품·AI 수요 호재 / 수출규제 노이즈 |
| 수출규제 뉴스 | AMD·AMAT | 중국 매출 비중 → 규제 강화 시 직접 타격 |
| AI capex 모멘텀 | ANET·NBIS | AI 데이터센터 투자 확대 = 수혜 |

### 종목별 민감도
```
NBIS  🔴🔴🔴  금리에 가장 민감 (PER 89배, 변동성 최고)
AMD   🔴🔴    금리 + 수출규제 + Computex 이벤트 다수
AMAT  🔴      금리 + 수출규제 직접 노출
ANET  🟡      AI capex 테마 연동, 상대적으로 안정
```
