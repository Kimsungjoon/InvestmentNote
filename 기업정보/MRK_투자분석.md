## 5. 최대 리스크 — 키트루다 특허 절벽 (2028)

```
2028년 : 키트루다 미국 핵심 특허 만료
2029년 : 바이오시밀러 경쟁 시작 + 미국 정부 IRA 약가 협상 발효
         → 연간 $250억+ 매출 위험에 노출
```

### Merck의 대응 전략

| 전략 | 내용 |
|------|------|
| 키트루다 피하주사(QLEX) | 2025년 9월 승인 → 환자 편의성 향상으로 처방 유지 |
| 파이프라인 $700억 투자 | 2030년대 중반까지 20개 이상 신약 출시 목표 |
| Winrevair | 폐동맥고혈압 신약, 승인 완료 → 차세대 주력 품목 |
| MK-0616 | 경구 PCSK9 억제제 (콜레스테롤), Phase 3 진행 중 |
| mRNA 암 백신 (V940) | 모더나와 공동 개발, Phase 3 진행 중 |
| ADC(항체-약물 복합체) | 켈룬·다이이찌산쿄 파트너십으로 차세대 항암제 개발 |

## 6. 투자 포인트 (긍정 요인)

### ✅✅ ROE 22~37% — S&P500 최상위권
- 자본 대비 수익 창출력이 압도적
- 키트루다의 높은 마진(영업이익률 ~32%)이 ROE를 견인

### ✅ PER 13.9배 — 특허 절벽 우려가 이미 반영
- 키트루다 2028 특허 만료 리스크가 주가에 선반영 → 저평가 상태
- 실제 매출 피해는 2029년부터 → 향후 2~3년 이익 성장 지속 가능

### ✅ 배당 수익률 ~3.5% + 자사주 매입
- 안정적인 FCF($170억)로 주주환원 지속
- 15년 이상 배당 성장 유지

### ✅ 파이프라인 $700억 — 키트루다 이후를 준비 중
- Winrevair, MK-0616, V940 등 다양한 성장 드라이버 준비 중
- M&A 활발: 시다라, 베로나, 프로메테우스 인수로 포트폴리오 다각화

## 7. 리스크 (부정 요인)

### ❌ 키트루다 의존도 49% — 단일 약품 집중 위험
- 키트루다 특허 만료(2028) 이후 바이오시밀러 진입 시 매출 급감 가능
- IRA(인플레이션 감소법)에 따른 정부 약가 협상: 2027년 선정, 2029년 발효

### ❌ PSR 4.6배 — 기준 초과
- 제약 섹터 특성이지만 밸류에이션 부담 존재

### ❌ 가다실 중국 리스크
- 중국 HPV 백신 시장 경쟁 심화 및 수요 불확실성

### ❌ 파이프라인 임상 실패 리스크
- $700억 파이프라인 중 상당수가 임상 단계 → 실패 시 주가 타격

## 9. 최종 투자 의견

### 🟢 투자 매력도 높음 — 단, 2028 이후 전략 모니터링 필수

**MRK의 핵심 논리**: 키트루다 특허 절벽(2028) 우려가 이미 주가에 반영되어  
ROE 37%, PER 13.9배라는 **빅파마 치고 이례적인 저평가** 상태입니다.

```
✅ 적극 매수 구간
  - 현재 주가 $115~125 → 특허 절벽 리스크 반영 후 저평가
  - 2026~2027: 키트루다 매출 지속 성장 + Winrevair 확대 기간
  - 파이프라인 임상 긍정 데이터 발표 시 주가 촉매

⚠️ 주의 구간
  - 2028년 하반기~2029년: 바이오시밀러 진입 시점
  - 파이프라인 핵심 후보 임상 실패 시 하방 압력

🎯 투자 전략
  - 장기 보유(3~5년)에 적합: 배당 3.5% 수취하며 파이프라인 성숙 대기
  - 2027 하반기까지 파이프라인 데이터 흐름 확인 후 비중 조정
```
