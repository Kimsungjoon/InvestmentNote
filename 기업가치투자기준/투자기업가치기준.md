# 투자 기업 선정 기준 (스윙 매매 기준)

> **투자 철학**: 펀더멘털이 탄탄한 기업 중에서, 기술적 진입 타이밍이 맞는 순간에만 매수한다.  
> 스윙 매매의 보유 기간 목표: **2주 ~ 2개월**

---

## 구조 개요

```
1단계: 펀더멘털 필터  →  망하거나 고평가된 기업 제외
2단계: 기술적 진입 기준  →  추세·모멘텀 확인 후 매수 타이밍 포착
3단계: 청산 기준  →  익절/손절 가격 미리 설정 후 실행
4단계: 리스크 관리  →  포지션 크기와 자산 배분 규칙
```

---

## 1단계 — 펀더멘털 필터 (진입 금지 종목 제거)

> 스윙 매매에서 펀더멘털은 "좋은 종목 고르기"가 아니라 **"나쁜 종목 제외하기"** 에 사용한다.
> 따라서 **역성장·과도한 부채·저성장 적자 기업만 거른다**. 성장주는 ROE가 낮아도 포용한다.

**필수 조건 (둘 다 충족)**

| 지표 | 기준 | 목적 |
|------|------|------|
| 부채비율 | **200% 이하** | 재무 위기 기업 제외 |
| 매출 추이 | **성장 중** (역성장 아님) | 사양 기업 제외 |

**수익성/성장 조건 (아래 중 하나 이상 충족)**

| 경로 | 기준 | 적용 대상 |
|------|------|----------|
| ① 수익성 | **ROE 10% 이상** | 탄탄한 수익 기업 |
| ② 흑자 | **순이익 흑자** (ROE 낮아도 OK) | 흑자 성장주 (소프트웨어·플랫폼 등) |
| ③ 초고성장 | **매출성장 30% 이상** (적자여도 OK) | 고성장 적자 기업 (우주·신성장) |

> **의도**: 반도체뿐 아니라 소프트웨어·우주·플랫폼 등 비반도체 성장주도 포용한다.
> 단, **PER은 가치 평가가 아닌 참고 지표로만 사용**한다 (고밸류는 기술적 진입·청산으로 관리).

## 2단계 — 기술적 진입 기준 (매수 타이밍)

> 아래 조건 중 **3개 이상 충족 시** 진입 검토

### 추세 확인 (대형 프레임)

| 조건 | 기준 | 의미 |
|------|------|------|
| 이동평균 위치 | **주가 > 50일선 > 200일선** | 중·장기 상승 추세 (배경 확인용) |
| 50일선 기울기 | **우상향** | 추세 강도 확인 |
| 52주 고점 대비 | **-30% 이내** | 추세 붕괴 종목 제외 |

### 단기 추세 확인 (단기 스윙 핵심)

> 보유 기간 **1~3주** 단기 스윙에서는 5일선·20일선을 진입 타이밍의 핵심으로 사용한다.

| 조건 | 기준 | 의미 |
|------|------|------|
| 5일선 위치 | **주가 > 5일선** | 현재 단기 매수세 우위 |
| 5일선 기울기 | **우상향** | 단기 모멘텀이 살아있음 |
| 5일선 vs 20일선 | **5일선이 20일선 위로 골든크로스** | 단기 추세 회복 신호 — 핵심 진입 시점 |
| 20일선 눌림 | **20일선 근처로 조정 후 반등** | 단기 눌림목 매수 기회 |

```
단기 스윙 이상적 진입 구조:

50일선·200일선 → 상승 추세 살아있음 확인 (배경)
20일선 근처까지 조정 발생               (눌림목)
5일선이 20일선 위로 회복·골든크로스      (진입 신호)
거래량 증가 확인                        (매수세 진짜인지 검증)
```

### 모멘텀 확인

| 조건 | 기준 | 의미 |
|------|------|------|
| RSI (14일) | **40 ~ 65 구간** | 과매도 회복 or 건전한 상승 모멘텀 |
| MACD | **시그널선 골든크로스 or 히스토그램 상승 전환** | 단기 모멘텀 반등 신호 |
| 거래량 | **20일 평균 거래량 이상** | 이동이 수반된 진입인지 확인 |

### 진입 타이밍

| 패턴 | 설명 |
|------|------|
| 지지선 반등 | 주요 지지선(50일선, 20일선, 전 저점)에서 반등 확인 후 진입 |
| 박스권 돌파 | 일정 기간 횡보 후 거래량 동반 상단 돌파 시 진입 |
| 눌림목 매수 | 상승 추세 중 20일선 조정 후 5일선 골든크로스 확인 시 진입 |
| 5일선 반등 | 단기 급락 후 5일선 위 복귀 + 거래량 증가 시 초단기 진입 |

## 3단계 — 청산 기준 (익절·손절)
### 손절 기준

### 고점 대비 손절 기준 (트레일링 스탑)

```
현재 수익률이 최고 수익률 대비 -10%p 이상 하락 시 즉시 청산
예) 최고 수익률 +25% → 현재 수익률 +15% 이하가 되면 청산
예) 최고 수익률 +8%  → 현재 수익률 -2% 이하가 되면 청산
```

- 수익을 지키는 가장 중요한 규칙
- 일반 손절(-7%)과 고점 대비 손절(-10%p) 중 **먼저 발동되는 쪽**을 따른다

### 익절 기준

| 상황 | 행동 |
|------|------|
| 수익률 +20% 도달 | 보유량의 **절반 익절**, 나머지는 손절가를 매수가(+0%)로 올려 관리 |
| 수익률 +30% 도달 | 나머지 전량 익절 |
| 수익률 +25% 이상에서 RSI 70 초과 | 과매수 신호 → 전량 익절 검토 |

### 시간 손절 (Time Stop)

```
매수 후 3주가 지나도 +5% 미달 시 → 절반 또는 전량 청산
```
→ 자금이 잠기는 기회비용을 막기 위한 규칙

### 리스크/리워드 계산 예시

```
매수가: $100
손절가: $93  (리스크: -$7)
목표가: $114 (리워드: +$14)
R/R 비율: 14 ÷ 7 = 2.0  →  진입 가능 ✅

목표가: $107 (리워드: +$7)
R/R 비율: 7 ÷ 7 = 1.0  →  진입 불가 ❌
```

## 4단계 — 리스크 관리

| 규칙 | 기준 |
|------|------|
| 종목당 최대 비중 | 개별 주식 포트폴리오의 **30% 초과 금지** |
| 동시 보유 종목 수 | 최대 **5종목** (집중하여 관리 가능한 수준) |
| 1회 손실 허용 한도 | 총 포트폴리오의 **1.5% 이내** (종목당 포지션 크기 계산 기준) |
| 나스닥 -3% 급락일 | 신규 매수 보류, 손절 기준 종목은 그대로 실행 |

## 요약 체크리스트

매수 전 아래 항목을 순서대로 확인한다.

```
[ ] 1. 펀더멘털 필터 통과? (ROE ≥ 10%, 부채비율 ≤ 200%, 매출 성장 중)
[ ] 2. 주가 > 50일선 > 200일선? (상승 추세 확인)
[ ] 3. RSI 40~65 구간? (과매수 아님)
[ ] 4. MACD 골든크로스 or 히스토그램 반등?
[ ] 5. 거래량 20일 평균 이상?
[ ] 6. 진입 패턴 해당? (지지선 반등 / 박스권 돌파 / 눌림목)
[ ] 7. R/R 비율 1:2 이상?
[ ] 8. 손절가·익절가 미리 설정했음?
```

> 8개 중 **6개 이상 ✅** 이면 매수 진행, 5개 이하면 다음 기회를 기다린다.
