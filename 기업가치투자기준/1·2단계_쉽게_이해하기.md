# 1단계·2단계 쉽게 이해하기

> 이 문서는 `투자기업가치기준.md`와 `screener.py`의 **1단계(펀더멘털)·2단계(기술적 진입)** 를
> 투자 처음 하는 사람도 이해할 수 있게 풀어 쓴 설명서입니다.
>
> **시장별로 전략이 다릅니다.**
> - 🇺🇸 **나스닥**: 추세추종 · 단기 스윙 (2주~2개월)
> - 🇰🇷 **코스피**: 가치투자 · 자산 방어 우선

---

## 들어가기 전에 — 큰 그림 한 장

우리 투자 방식은 **"두 개의 문"**을 차례로 통과시키는 겁니다.

```
        모든 종목
           │
   ┌───────▼────────┐
   │  1단계 문       │   "이 회사, 후보로 쓸 만하지?"   ← 회사 체력 / 안전마진
   │  (펀더멘털)     │
   └───────┬────────┘
           │  통과한 종목만
   ┌───────▼────────┐
   │  2단계 문       │   "지금이 살 타이밍 맞아?"        ← 차트·수급 타이밍
   │  (기술적 진입)  │
   └───────┬────────┘
           │
        매수 검토
```

| | 나스닥 | 코스피 |
|--|--------|--------|
| 1단계 목적 | 빚 많거나 너무 작은 종목만 걸러냄 | 안전마진·재무·실적 점수화 |
| 2단계 목적 | 눌림목 / 돌파 / 에너지응축 타이밍 | 20MA·RSI·기관/외인 수급으로 가점 |
| 스크리너 | `python3 screener.py` | `python3 screener.py --kospi` |

> 핵심 비유: **1단계 = 메뉴판에서 먹을 만한 식당 고르기**, **2단계 = 줄 안 서고 들어갈 타이밍**.

# 🇺🇸 나스닥 — 추세추종 스윙

## 1단계 — 기본 필터 (나쁜 종목 제외)

스윙에서는 **"최고의 회사 고르기"보다 "지뢰 피하기"**가 중요합니다.
스크리너는 나스닥에서 아래만 **필수**로 봅니다.

| 조건 | 기준 | 쉬운 설명 |
|------|------|----------|
| 시총 | **$10B 이상** | 너무 작은 종목(유동성·조작 위험) 제외 |
| 부채비율 | **200% 이하** | 재무 위기 수준 제외 (데이터 없으면 통과) |
| ROE | **7% 이상** (미확인 시 탈락) | 수익성 확인 |
| 매출성장 | **5% 이상** | 성장 정체 제외 |
| PEG | **2.5 이하** | forward PER ÷ 성장률 (없으면 trailing) |
| PER | **80 이하** | **trailing PER** 절대 상한 |
| 영업이익률 | **> 0** (확인 시) | 적자 기업 제외 |

### ROE·PER·PEG는요?

- 성장주는 PER 절대값만 보면 항상 비싸 보입니다 → **성장·ROE·PEG(forward 우선)** 로 판단합니다.
- **trailing PER 80 초과**는 무조건 제외 — forward PEG가 좋아도 안전망 유지.
- 차트 조건만 맞고 펀더가 안 맞으면 **「기술 통과·펀더 미달」**로 따로 표시됩니다.

```
나스닥 1단계 통과 =
  시총 ≥ $10B  AND  부채비율 ≤ 200%(또는 N/A)
  AND  ROE ≥ 7%  AND  매출성장 ≥ 5%  AND  forwardPEG ≤ 2.5  AND  trailingPER ≤ 80
```

## 2단계 — 기술적 진입 (살 타이밍)

### 먼저 확인할 것 — 시장 레짐 (QQQ)

```
QQQ > 50일선  AND  50일선이 우상향  → 🟢 강세 (진입 허용)
아니면                                 → 🔴 약세 (후보 표시는 하되, 실제 매수는 보류 권장)
```

개별 종목 차트가 좋아도 **지수 전체가 약하면** 성공 확률이 떨어집니다.

### 이동평균선이 뭔가요?

```
5일선   = 아주 단기 (가장 민감)
20일선  = 단기 (약 한 달)
50일선  = 중기 (약 두세 달)
200일선 = 장기 (약 1년)
```

주가가 이평선 **위** = 그 기간 산 사람들이 이익 중 → 분위기 좋음.

### 공통 추세 조건 (진입 유형 분류 전 필수)

| 조건 | 기준 |
|------|------|
| 주가 > 50일선 | 중기 상승 추세 |
| 50일선 우상향 | 중기 기울기 살아있음 |

이게 안 되면 → 태그 `추세X`, 후보에서 제외.

---

### 진입 유형 3가지 (스크리너 핵심)

스크리너는 추세 OK인 종목을 아래 **셋 중 하나**로 분류합니다.

#### ① 눌림목 — "잠시 쉬다 다시 오를 자리"

| 조건 | 기준 |
|------|------|
| 5MA 이격 | **±2.5% 이내** (5일선에 바짝) |
| 20MA 거리 | 5-20MA 이격이 **-4%보다 위에** (너무 깊게 안 깨짐) |
| RSI | **38 ~ 65** |
| 거래량 | 5일/20일 비율 **≤ 1.05** (수축 = 매도 소진) |

```
오르다 → 5일선 근처로 살짝 쉬기 → 거래량 줄며 버티기 → 매수

      /\      /\
     /  \    /   ← 여기서 또 오름
    /    \  /
   /      \/  ← 눌림목 (5MA·20MA 근처)
```

#### ② 돌파 — "신고가·박스 상단을 뚫는 자리"

| 조건 | 기준 |
|------|------|
| 고점 대비 | **52주 고점 -6% 이내** |
| RSI | **58 ~ 82** |
| 거래량 | 5일/20일 **≥ 1.1** (급증 동반) |
| 이평선 | **200MA 위 + 20MA 위** |

```
옆으로 횡보 or 고점 근처 → 거래량 늘며 위로 뚫음 → 매수
```

#### ③ 에너지응축 — "弹簧처럼 모아진 자리"

| 조건 | 기준 |
|------|------|
| 5-20MA 간격 | **\|5MA−20MA\| ≤ 5%** |
| 20-50MA 간격 | **\|20MA−50MA\| ≤ 8%** |
| 20일 가격 폭 | **≤ 14%** |
| RSI | **43 ~ 70** |

```
5·20·50일선이 서로 바짝 모임 = 곧 한쪽으로 크게 움직일 준비
상방 돌파 확인되면 진입
```

### QQQ 상대강도 (RS)

| 표시 | 의미 |
|------|------|
| **RS▲** | 최근 20일 수익률이 QQQ보다 **강함** → 주도주 쪽 |
| **RS▼** | QQQ보다 **약함** → 후순위 |

스윙에서는 같은 눌림목이라도 **RS▲를 우선**합니다.

### 나스닥 2단계 한 줄 정리

```
[레짐] QQQ 50MA 위 강세이고
[추세] 주가 > 50MA + 50MA 우상향이며
[유형] 눌림목 or 돌파 or 에너지응축 중 하나면
→ 2단계 통과 (= 매수 후보)
```

# 🇰🇷 코스피 — 가치투자 · 자산 방어

## 1단계 — 안전마진 · 재무 (필수 탈락)

코스피는 **"싸게 사서 지키는"** 관점이 우선입니다.
아래 **하나라도** 해당하면 점수와 무관하게 **조건 미달**.

| 기준 | 탈락 조건 |
|------|-----------|
| 부채비율 | **> 200%** |
| ROE | **< 7%** (데이터 있을 때) |
| PBR | **> 1.5** |
| PER | **> 22** 또는 적자(PER&lt;0) |

### 용어 쉬운 설명

| 지표 | 뜻 | 왜 보나 |
|------|----|--------|
| **PBR** | 주가 ÷ 순자산 | 1 근처면 자산 대비 비싸지 않음 (안전마진) |
| **PER** | 주가 ÷ 이익 | 너무 높으면 기대만 반영된 고평가 |
| **ROE** | 자기자본으로 돈 버는 효율 | 7% 미만이면 체력이 약함 |
| **목표주가 괴리율** | (증권사 목표가 ÷ 현재가) − 1 | 15% 이상이면 상승 여력 가점 |

---

## 2단계 — 10점 점수화 (우선 / 관찰)

스크리너가 **10점 만점**으로 점수를 매깁니다.

| 영역 | 항목 | 점수 |
|------|------|:----:|
| 가치 (최대 3) | PBR ≤ 1.0 | +2 |
| | PBR 1.0~1.5 | +1 |
| | PER ≤ 12 | +1 |
| 실적 (최대 4) | 매출성장 > 0% | +1 |
| | 영업이익률 > 0% | +1 |
| | 목표주가 괴리율 ≥ 15% | +2 |
| 수급·타이밍 (최대 3) | 주가 > 20MA | +1 |
| | RSI 38~70 | +1 |
| | 기관+외국인 5일 누적 순매수 > 0 (네이버) | +1 |

| 점수 | 분류 |
|:----:|------|
| **7점 이상** | ✅ **우선 후보** — 매수 검토 1순위 |
| **5~6점** | ⬜ **관찰 후보** — 추가 확인 후 검토 |
| **4점 이하** 또는 필수 탈락 | ❌ **조건 미달** |

### 코스피에서 차트는?

나스닥처럼 "골든크로스만 보고 산다"가 아니라,
**저평가·실적·수급이 갖춰진 종목**에 타이밍 가점을 주는 구조입니다.
수급은 Yahoo가 아니라 **네이버 금융 일별 기관·외국인 순매매량**으로 집계합니다.
(거래량 비율은 수급 대용이 아닙니다. 장중 당일 수급은 미확정일 수 있음.)

### 코스피 한 줄 정리

```
[필수] 빚·ROE·PBR·PER이 극단 고위험 아니고
[점수] 가치 + 실적 + 수급/타이밍 합계 7점 이상이면
→ 우선 후보
```

# 전체를 하나로 — 매수 결정 흐름

## 나스닥

```
[1단계] 시총·단가·부채 OK?
        │ 통과
[2단계] QQQ 강세? → 주가>50MA? → 눌림목/돌파/응축?
        │ 통과
   목표가·손절가·손익비 확인 후 매수
```

## 코스피

```
[1단계] 부채·ROE·PBR·PER 탈락 아님?
        │ 통과
[2단계] 10점 점수 ≥ 7? (또는 관찰 5~6)
        │ 통과
   목표가 괴리·수급(HTS)·리스크 확인 후 매수
```

---

## 자주 하는 오해 3가지

```
오해 1) "좋은 회사면 아무 때나 사도 된다"
   → ❌ 나스닥은 타이밍(2단계)이 핵심. 과열·추세X에서 사면 물린다.

오해 2) "많이 떨어졌으니 싸다"
   → ❌ 나스닥: 고점 -30% 이상은 '눌림'이 아니라 '추세 훼손'에 가깝다.
   → ❌ 코스피: 낙폭만으로 사지 말고 PBR·실적·수급을 함께 본다.

오해 3) "스크리너 나왔으면 바로 사자"
   → ❌ 스크리너는 후보 압축 도구. 뉴스·실적·손익비는 사람(또는 최종분류 프롬프트)이 확인.
```
