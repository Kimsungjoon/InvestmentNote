# 1단계·2단계 쉽게 이해하기

> 이 문서는 `투자기업가치기준.md`의 **1단계(펀더멘털 필터)**와 **2단계(기술적 진입 기준)**를
> 투자 처음 하는 사람도 이해할 수 있게 풀어 쓴 설명서입니다.

---

## 들어가기 전에 — 큰 그림 한 장

우리 투자 방식은 **"두 개의 문"**을 차례로 통과시키는 겁니다.

```
        모든 종목
           │
   ┌───────▼────────┐
   │  1단계 문       │   "이 회사, 망할 회사는 아니지?"   ← 회사의 체력 검사
   │  (펀더멘털)     │
   └───────┬────────┘
           │  통과한 종목만
   ┌───────▼────────┐
   │  2단계 문       │   "지금이 살 타이밍 맞아?"        ← 주가의 타이밍 검사
   │  (기술적 진입)  │
   └───────┬────────┘
           │
        매수 검토
```

- **1단계**는 "**어떤 회사**를 후보로 둘까"를 정합니다. (회사의 펀더멘털 = 기초 체력)
- **2단계**는 "**언제** 살까"를 정합니다. (주가 차트의 타이밍)

> 핵심 비유: **1단계 = 좋은 식당 고르기**, **2단계 = 줄 안 서고 들어갈 타이밍 고르기**.
> 아무리 좋은 식당이어도 2시간 줄 서면 별로고(=비싼 타이밍), 맛없는 식당은 줄이 없어도 안 갑니다(=나쁜 회사).

---

# 1단계 — 펀더멘털 필터 (회사 체력 검사)

## 핵심 사고방식

스윙 매매에서 펀더멘털은 **"최고의 회사를 고르는 도구"가 아닙니다.**
**"망할 회사를 걸러내는 도구"**입니다.

```
❌ (가치투자) "가장 저평가된 보석 같은 회사를 찾자"
✅ (스윙매매) "지뢰(망할 회사)만 안 밟으면 된다. 나머지는 차트로 고른다"
```

그래서 기준이 **느슨**합니다. 나쁜 것만 거르고 나머지는 다 통과시킵니다.

---

## 통과 조건 — "필수 2개" + "성장성 1개"

### 필수 조건 (이 둘은 무조건 충족해야 함)

#### ① 부채비율 250% 이하

**부채비율이란?** 회사가 가진 자기 돈(자본) 대비 빚이 얼마나 많은가입니다.

```
부채비율 = (빚 ÷ 자기자본) × 100

예) 자기 돈 100억, 빚 50억  → 부채비율 50%   (안전)
    자기 돈 100억, 빚 300억 → 부채비율 300%  (위험! 탈락)
```

- **왜 보나?** 빚이 너무 많으면 금리가 오르거나 매출이 흔들릴 때 **휘청거리다 망할 수 있어서**입니다.
- **250%는 느슨한 편**입니다. 정말 위험한 회사만 거르겠다는 뜻입니다.

#### ② 매출이 성장 중 (역성장 아님)

```
✅ 작년보다 매출이 늘었다              → 통과
❌ 매출이 계속 줄고 있다 (사양산업)    → 탈락
```

- **왜 보나?** 매출이 줄어드는 회사는 **사업이 시들고 있다**는 신호입니다. 주가도 장기적으로 약합니다.

---

### 성장성 조건 (아래 3가지 중 **하나만** 충족하면 OK)

여기가 이번에 **느슨하게 바꾼 핵심**입니다. 회사마다 성격이 다르니 **3개의 길** 중 하나만 통과하면 됩니다.

| 경로 | 기준 | 어떤 회사인가 | 쉬운 설명 |
|------|------|--------------|----------|
| ① **수익성** | ROE 10% 이상 | 돈 잘 버는 알짜 회사 | "이미 돈을 잘 굴린다" |
| ② **흑자** | 순이익 흑자 | 흑자 내는 성장주 | "큰 부자는 아니어도 흑자다" |
| ③ **초고성장** | 매출성장 30% 이상 | 적자라도 폭풍 성장 | "지금 적자지만 미친 듯이 큰다" |

#### ROE가 뭔가요? (경로 ①)

**ROE = 자기 돈으로 얼마나 효율적으로 버는가** (Return On Equity, 자기자본이익률)

```
ROE = (순이익 ÷ 자기자본) × 100

예) 내 돈 100억 넣어서 1년에 15억 벌었다  → ROE 15%
    내 돈 100억 넣어서 1년에 3억 벌었다    → ROE 3%
```

- **ROE 10% 이상** = "내 돈 100만원 맡기면 1년에 10만원 이상 불려준다" 수준. 꽤 잘하는 회사.

#### 왜 "3개의 길"로 나눴나요?

성장주(특히 소프트웨어·우주·신기술)는 **번 돈을 미래에 재투자**하느라 당장 ROE가 낮거나 적자입니다.
이걸 ROE 하나로만 자르면 **테슬라 초창기, 아마존 초창기 같은 회사를 다 놓칩니다.**

```
예) NBIS(AI 클라우드): ROE 14% + 매출성장 684%  → 경로 ③(초고성장)로 통과
    AMD: ROE 8%(기준 미달)지만 흑자 + 매출성장 38% → 경로 ②·③로 통과
    인텔: 적자 + 매출성장 7%(저성장)            → 3개 길 모두 실패 → 탈락
```

> **요약**: 돈 잘 벌거나(①), 흑자거나(②), 폭풍 성장하거나(③) — **셋 중 하나면 합격.**
> 단, **적자인데 성장도 느린 회사**는 어느 길로도 못 가서 탈락합니다. (예: 인텔)

---

## PER은요?

**PER = 주가가 이익의 몇 배인가** (비쌈/쌈의 척도). 예: PER 30 = 회사가 버는 이익의 30배 가격.

- 예전엔 "PER 50 이하"를 요구했지만, **지금은 참고만 합니다.**
- 이유: 고성장주는 원래 PER이 높습니다(미래 기대가 가격에 미리 반영). 비싼지 싼지는 **차트(2단계)로 관리**합니다.

---

## 1단계 한 줄 정리

```
[필수] 빚 너무 많지 않고(부채 250%↓) + 매출 줄지 않으면서
[성장] 돈 잘 벌거나 / 흑자거나 / 폭풍 성장 중 하나면
→ 1단계 통과 (= 후보 등록)
```

---

# 2단계 — 기술적 진입 기준 (살 타이밍 검사)

1단계를 통과해도 **"아무 때나"** 사면 안 됩니다.
좋은 회사라도 **이미 많이 올라 비싸진 자리**에서 사면 물립니다.
2단계는 **차트를 보고 "지금이 살 자리인지"**를 확인합니다.

> **규칙**: 아래 조건들 중 **여러 개가 동시에 ✅** 일 때만 진입.

---

## 먼저 — 이동평균선(이평선)이 뭔가요?

**이동평균선 = 최근 며칠간 종가의 평균을 이은 선.** 주가의 "평균적인 흐름"을 보여줍니다.

```
5일선   = 최근 5일 종가 평균   → 아주 단기 흐름 (가장 민감, 출렁임)
20일선  = 최근 20일 평균       → 단기 흐름 (약 한 달)
50일선  = 최근 50일 평균       → 중기 흐름 (약 두세 달)
200일선 = 최근 200일 평균      → 장기 흐름 (약 1년, 큰 추세)
```

**핵심 직관**:
- 주가가 이평선 **위**에 있으면 → 그 기간 산 사람들이 **이익 중** = 분위기 좋음(상승)
- 주가가 이평선 **아래**에 있으면 → 그 기간 산 사람들이 **손실 중** = 분위기 나쁨(하락)

---

## 2단계를 3개의 층으로 나눠서 봅니다

```
[층1] 큰 추세    : 지금 이 종목, 큰 그림이 상승장인가?   (50일선·200일선)
[층2] 단기 타이밍: 지금이 단기 매수 자리인가?            (5일선·20일선)  ← 단기 스윙 핵심
[층3] 힘 확인    : 이 움직임에 진짜 힘이 실렸나?          (RSI·MACD·거래량)
```

---

### [층1] 큰 추세 확인 — "상승장 배경인가?"

| 조건 | 기준 | 쉬운 설명 |
|------|------|----------|
| 이평선 정배열 | 주가 > 50일선 > 200일선 | 단기·중기·장기가 위에서부터 차곡차곡 = 건강한 상승 |
| 50일선 기울기 | 우상향 | 중기 흐름이 위를 향함 |
| 52주 고점 대비 | -30% 이내 | 고점에서 너무 폭락한 종목(추세 붕괴)은 제외 |

**정배열 그림**:
```
   주가      ← 맨 위
    └ 50일선
       └ 200일선  ← 맨 아래
이렇게 위→아래 순서면 "정배열" = 상승 추세 (좋음)

반대로 200일선이 맨 위면 "역배열" = 하락 추세 (피함)
```

> 이 층은 **"배경 확인용"**입니다. 큰 그림이 상승이어야 단기 매수가 성공할 확률이 높습니다.

---

### [층2] 단기 타이밍 — "지금 들어갈 자리인가?" (단기 스윙의 핵심)

1~3주 들고 가는 단기 스윙에서는 **5일선과 20일선**이 가장 중요합니다.

| 조건 | 기준 | 쉬운 설명 |
|------|------|----------|
| 주가 > 5일선 | 주가가 5일선 위 | 지금 당장의 단기 매수세가 우위 |
| 5일선 우상향 | 5일선이 올라가는 중 | 단기 힘이 살아있음 |
| 5일선 > 20일선 (골든크로스) | 5일선이 20일선을 위로 뚫음 | **단기 추세 회복 신호 — 핵심 진입 시점** |
| 20일선 눌림 | 20일선까지 내렸다가 반등 | 잠깐 쉬었다 다시 오르는 "눌림목" |

#### "골든크로스"가 뭔가요?

```
짧은 선(5일선)이 긴 선(20일선)을 아래에서 위로 뚫고 올라가는 것
= 단기 분위기가 "하락 → 상승"으로 바뀌는 신호
= 사기 좋은 타이밍

반대(5일선이 20일선을 아래로 뚫음) = 데드크로스 = 팔 신호
```

#### "눌림목"이 뭔가요?

```
계단 오르듯 주가가 오를 때, 잠깐 쉬며 20일선 근처까지 살짝 내려오는 구간.
이때가 "싸게 올라타는" 자리.

      /\      /\
     /  \    /   ← 여기서 또 오름
    /    \  /
   /      \/  ← 눌림목 (20일선 근처, 매수 기회)
  /
```

#### 단기 스윙의 "이상적인 진입 그림"

```
1) 50일선·200일선 위 → 큰 추세는 상승  (층1 확인)
2) 20일선 근처까지 잠깐 조정          (눌림목 발생)
3) 5일선이 20일선 위로 골든크로스      (진입 신호 점등!)  ← 여기서 산다
4) 거래량이 늘어남                    (진짜 매수세 확인)
```

---

### [층3] 힘 확인 — "이 움직임, 진짜 힘이 실렸나?"

차트가 올라도 **힘없이 오르면** 금방 꺾입니다. 아래 3개로 힘을 확인합니다.

#### ① RSI (40~65 구간)

**RSI = 과열/냉각 온도계** (0~100). 최근 상승·하락의 강도를 측정.

```
RSI 70 이상 → 과열 (너무 많이 올라 사람들이 흥분 상태) = 추격 위험
RSI 40~65  → 적당히 건강한 상승 or 과매도에서 회복 중 = 좋은 진입 구간 ✅
RSI 30 이하 → 과냉 (공포에 너무 팔림)
```

- **왜 70 이상을 피하나?** 이미 많이 올라서, 살짝만 나쁜 소식 와도 **확 빠지기** 때문. (고점에 물림)

#### ② MACD (골든크로스 / 히스토그램 상승)

**MACD = 단기 흐름과 중기 흐름의 힘 차이.** 모멘텀(추진력)이 살아나는지 봅니다.

```
MACD 히스토그램이 ▼(하락) → ▲(상승)으로 바뀌면
= 떨어지던 힘이 다시 오르기 시작 = 반등 신호
```

#### ③ 거래량 (20일 평균 이상)

**거래량 = 그날 거래된 주식 수 = 시장의 관심도.**

```
주가가 오르는데 거래량도 같이 늘었다 → 진짜 매수세 (믿을 만함) ✅
주가는 오르는데 거래량이 적다        → 힘없는 상승 (속을 수 있음) ⚠️
```

- 비유: **많은 사람이 동의하며 미는 것**(거래량 ↑)과 **혼자 미는 것**(거래량 ↓)의 차이.

---

## 2단계 진입 패턴 4가지 (실전 모양)

| 패턴 | 그림으로 보면 | 진입 방법 |
|------|--------------|----------|
| **지지선 반등** | 내려오다 50/20일선에서 톡 튕김 | 반등 확인 후 진입 |
| **박스권 돌파** | 한참 옆으로 횡보하다 위로 뚫음 | 거래량 동반 돌파 시 진입 |
| **눌림목 매수** | 오르다 20일선까지 쉬고 다시 골든크로스 | 5일선 골든크로스 확인 시 진입 |
| **5일선 반등** | 급락 후 5일선 위로 빠르게 복귀 | 거래량 늘며 복귀 시 초단기 진입 |

---

## 2단계 한 줄 정리

```
[배경] 큰 추세가 상승(주가>50>200)이고
[타이밍] 5일선이 20일선 위로 회복(골든크로스)했으며, 눌림목 자리고
[힘] RSI 40~65(과열 아님) + 거래량 동반이면
→ 2단계 통과 (= 지금 사도 되는 자리)
```

---

# 전체를 하나로 — 매수 결정 흐름

```
[1단계] 이 회사 망할 회사 아니지?
   ├─ 빚 250% 이하 ✅
   ├─ 매출 성장 중 ✅
   └─ 돈 잘 벎/흑자/폭풍성장 중 하나 ✅
        │ 통과
        ▼
[2단계] 지금이 살 타이밍 맞아?
   ├─ 큰 추세 상승 (주가>50>200) ✅
   ├─ 5일선>20일선 골든크로스 ✅
   ├─ RSI 40~65 (과열 아님) ✅
   └─ 거래량 동반 ✅
        │ 통과
        ▼
   매수! (단, 손절가·익절가 미리 정하고 — 3단계)
```

## 실제 예시로 이해하기 (2026-05-30 기준)

| 종목 | 1단계 | 2단계 | 종합 판단 |
|------|------|------|----------|
| **AMAT** | ✅ (ROE 39.7%) | ✅ (RSI 55, 5일선 위, 정배열) | 회사 좋고 타이밍 좋음 → **매수 적격** |
| **인텔** | ❌ (적자+저성장) | ✅ (RSI 43, 자리는 OK) | **회사가 1단계 탈락** → 자리 좋아도 패스 |
| **GEV** | (해당없음) | ❌ (5·20일선 아래로 이탈) | 추세 꺾임 → **진입 금지** |
| **DELL** | ✅ | ❌ (RSI 84, 과열) | 회사는 좋으나 **너무 비싼 자리** → 눌림 대기 |

> **교훈**:
> - 회사 좋아도 타이밍 나쁘면(=DELL) 기다린다.
> - 타이밍 좋아도 회사 나쁘면(=인텔) 안 산다.
> - **둘 다 ✅일 때만** 산다. → 그게 AMAT 같은 케이스.

---

## 자주 하는 오해 3가지

```
오해 1) "좋은 회사면 아무 때나 사도 된다"
   → ❌ 비싼 자리에 사면 좋은 회사여도 물린다 (2단계가 필요한 이유)

오해 2) "많이 떨어졌으니 싸다, 사자"
   → ❌ 떨어지는 칼날. RSI·이평선으로 '바닥 확인 후' 진입 (눌림 ≠ 추락)

오해 3) "RSI 80인데 더 오를 것 같아"
   → ❌ 과열 구간 추격은 스윙에서 가장 자주 물리는 실수
```
