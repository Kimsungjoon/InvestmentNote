# 가치 기준 충족 미국 주식 추천 (섹터별)

> **주의**: 아래 수치는 2025년 기준 추정치이며, 주가는 매일 변합니다.
> 실제 투자 전 [Finviz](https://finviz.com), [Macrotrends](https://www.macrotrends.net), [Simply Wall St](https://simplywall.st) 등에서 **최신 수치를 반드시 직접 확인**하세요.

---

## 선정 기준 요약

| 지표 | 기준 |
|------|------|
| ROE | 꾸준히 15% 이상 |
| PER | 15배 이하 |
| PSR | 3배 이하 (1.5배 이하 베스트) |
| PBR | 1.5배 이하 베스트 |
